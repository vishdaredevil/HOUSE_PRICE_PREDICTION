# House Price Prediction

This project predicts house prices using Linear Regression.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Steps
1. Data loading using Pandas
2. Feature selection (area, rooms)
3. Train-test split
4. Model training using Linear Regression
5. Prediction
6. Visualization
7. Model evaluation

## Conclusion
The model predicts house prices based on input features like area and number of rooms.
